# frontend/src/components/

Reusable UI components — **Stage 2 (React migration)**.

## Planned components

| Component | Description |
|-----------|-------------|
| `SchemeCard.jsx` | Single scheme card with match score bar |
| `Sidebar.jsx` | Navigation sidebar with user profile mini card |
| `Modal.jsx` | Onboarding / profile modal wrapper |
| `ChatBubble.jsx` | Single chat message (AI or user) |
| `DeadlineItem.jsx` | Single deadline row with urgency colour |
| `MatchBar.jsx` | Animated eligibility match percentage bar |
| `QuickReply.jsx` | Chat quick-reply pill button |
| `StatCard.jsx` | Dashboard stat card |

## Stage 1 note

In Stage 1, all component HTML is rendered via template strings in `index.html`.
These files will be created when Stage 2 migration to React begins.
