# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Stage 1 (current) | ✅ |

## Reporting a Vulnerability

EduAI is a static frontend application in Stage 1. It does not collect user data, store credentials, or connect to any backend.

If you discover a security vulnerability — for example, an XSS issue in the chat interface — please **do not open a public GitHub issue**.

Instead, email us at: `security@eduai.in` (or open a private advisory on GitHub)

We will acknowledge within 48 hours and aim to resolve within 7 days.

**Please include:**
- A description of the vulnerability
- Steps to reproduce
- Potential impact

Thank you for helping keep EduAI safe.
